# projeto-casa-do-dragao
🐉 Dragões da Casa do Dragão Projeto interativo desenvolvido para aprofundar conhecimentos em Manipulação de DOM, CSS Responsivo e Lógica com JavaScript. Este projeto é uma evolução do desafio proposto pelos irmãos Ricardo e Roberto (canal Dev em Dobro), com expansão de conteúdo e novas funcionalidades de interface.

### 🔗 Link do projeto => https://mhenrique89.github.io/projeto-casa-do-dragao/

### 🛠 Tecnologias e Conceitos Utilizados neste projeto.
<p align="center"> <img src="https://skillicons.dev/icons?i=html,css,js,vscode"/> </p>

HTML5 (Semântica e estrutura)

CSS3 (Flexbox, Media Queries, Transições e Animações)

JavaScript (Manipulação de classes, Event Listeners e lógica de carrossel)

## 🚀 O que foi desenvolvido
Expandido o elenco: Adicionados novos dragões da série A Casa do Dragão (totalizando 10+ dragões).

Organização por facções: Implementada lógica para separar dragões entre "Pretos" e "Verdes".

Feedback Visual Temático: O site altera dinamicamente a borda e o estilo visual conforme o dragão selecionado (atribuição de facção).
